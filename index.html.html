ਗੁਰਮੁਖੀ ਵਰਣਮਾਲਾ — The Gurmukhi Alphabet
Punjabi is written in the Gurmukhi script. There are 35 letters arranged in groups by the place in your mouth where each sound is made. Learn the name, sound and a word example for each letter!

ਮੁਹਾਰਨੀ — The Muharni
The Muharni combines each consonant with the 10 Gurmukhi vowel signs (laga matra). Reading across each row gives you all the vowel sounds on one letter. This is how Punjabi children traditionally learn to read — row by row!
